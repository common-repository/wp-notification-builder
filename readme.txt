== Description ==
WordPress Notification Builder you can build fully custom notification for your website any time anywhere. 
You can set special notification easily and customize as you like. Easy to use and you can create notification message with title and message body.


== Features ==
* Write Notification Title
* Write notification message with image our you can custom url show.
* Change Title Text color.
* You Can Change Title Background Color.
* You Can change Body Text Color.
* You Can change Body Background color. 
* Customize The Overly Color
* Saves all changelog in cache to keep things optimized

== Installation ==
To add a WordPress Plugin using the built-in plugin installer:

Go to Plugins > Add New.

1. Type in the name \"Wordpress Notification Builder\" in Search Plugins box
2. Find the \"Wordpress Notification Builder\" Plugin you wish to install.
3. Click Install Now to install the WordPress Plugin.
4. The resulting installation screen will list the installation as successful or note any problems during the install.
If successful, click Activate Plugin to activate it, or return to plugin installer for further actions.
